# AI简历筛选系统 - 设计系统规范

## 设计风格概述

本系统采用简洁、专业的设计风格，以提高用户的工作效率和使用体验。设计注重信息的清晰展示和操作的便捷性，符合企业级应用的专业性要求。

## 颜色系统

### 主色

- 主色：`#1890FF` - 用于主要按钮、重点链接和关键操作元素
- 主色-浅色：`#E6F7FF` - 用于主色元素的背景、悬停状态
- 主色-深色：`#0050B3` - 用于主色元素的激活状态

### 辅助色

- 成功色：`#52C41A` - 表示成功状态、积极结果
- 警告色：`#FAAD14` - 表示警告状态、需要注意
- 错误色：`#F5222D` - 表示错误状态、严重问题
- 信息色：`#1890FF` - 表示普通信息提示

### 中性色

- 标题文字：`#262626` - 用于页面主标题、重要文本
- 正文文字：`#595959` - 用于页面正文内容
- 次要文字：`#8C8C8C` - 用于辅助说明文字
- 禁用文字：`#BFBFBF` - 用于禁用状态的文字
- 边框颜色：`#D9D9D9` - 用于边框、分割线
- 背景色-浅：`#F5F5F5` - 用于页面背景、卡片背景
- 背景色-深：`#FAFAFA` - 用于表格斑马纹、次要背景

## 字体系统

### 字体家族

- 中文字体：`-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif`
- 数字字体：`-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif`

### 字体大小

- 特大标题：`24px` - 用于页面主标题
- 大标题：`20px` - 用于页面次级标题
- 中标题：`16px` - 用于卡片标题、区块标题
- 小标题：`14px` - 用于表格标题、列表标题
- 正文：`14px` - 用于正文内容
- 辅助文字：`12px` - 用于辅助说明、标签

### 行高

- 标题行高：`1.5` - 用于各级标题
- 正文行高：`1.5` - 用于正文内容

## 间距系统

基于8px的间距系统，提供一致的间距控制：

- 超小间距：`4px` - 用于紧凑元素内部间距
- 小间距：`8px` - 用于相关元素之间的间距
- 中间距：`16px` - 用于组件之间的间距
- 大间距：`24px` - 用于区块之间的间距
- 超大间距：`32px` - 用于主要区域之间的间距

## 阴影

- 卡片阴影：`0 2px 8px rgba(0, 0, 0, 0.09)` - 用于卡片、弹窗
- 浮层阴影：`0 4px 12px rgba(0, 0, 0, 0.12)` - 用于下拉菜单、悬浮提示

## 圆角

- 小圆角：`2px` - 用于标签、小型按钮
- 中圆角：`4px` - 用于卡片、按钮、输入框
- 大圆角：`8px` - 用于模态框、大型卡片

## 基础组件

### 按钮

- 主要按钮：蓝色背景，白色文字，用于页面主要操作
- 次要按钮：白色背景，蓝色边框和文字，用于次要操作
- 文本按钮：无背景无边框，蓝色文字，用于辅助操作
- 危险按钮：红色样式，用于危险操作
- 尺寸：大(高40px)、中(高32px)、小(高24px)

### 输入框

- 基础输入框：带有标签和提示文本
- 搜索框：带有搜索图标
- 文本域：用于多行文本输入
- 数字输入框：带有增减按钮
- 状态：默认、悬停、聚焦、禁用、错误

### 卡片

- 基础卡片：白色背景，带有标题和内容区域
- 简单卡片：无标题，仅内容区域
- 带操作卡片：底部带有操作按钮

### 表格

- 基础表格：带有表头和数据行
- 斑马纹表格：奇偶行不同背景色
- 可排序表格：表头可点击排序
- 可选择表格：行可选择
- 可展开表格：行可展开显示详情

### 标签

- 基础标签：不同颜色区分不同类型
- 可关闭标签：带有关闭按钮
- 状态标签：表示不同状态（成功、处理中、警告、错误）

## 响应式设计

### 断点

- 手机：`< 576px`
- 平板：`≥ 576px`
- 桌面小屏：`≥ 992px`
- 桌面大屏：`≥ 1200px`

### 栅格系统

采用24列栅格系统，根据不同断点响应式调整布局：

- 手机：通常使用1-2列布局
- 平板：通常使用2-4列布局
- 桌面小屏：通常使用4-6列布局
- 桌面大屏：通常使用6-8列布局

## 图标系统

- 线性图标：2px描边，用于界面元素
- 填充图标：用于强调和状态指示
- 尺寸：小(16px)、中(20px)、大(24px)

## 动效

- 过渡动画：`0.2s cubic-bezier(0.645, 0.045, 0.355, 1)`
- 加载动画：旋转、脉冲效果
- 交互反馈：点击涟漪、悬停变色

## 1. 用户认证流程

### 1.1 登录流程

```
用户 -> 访问登录页面
  |
  v
选择登录方式 --> 账号密码登录 --> 输入用户名/密码 --> 提交
  |                                                  |
  v                                                  v
第三方登录 --> 选择第三方平台 --> 授权             验证凭据
  |                                |                  |
  v                                v                  v
扫码登录 --> 扫描二维码           重定向回系统      成功/失败
                |                                     |
                v                                     v
              授权                                  失败 --> 显示错误信息
                |                                     |
                v                                     v
              重定向回系统                          成功 --> 保存登录状态
                                                      |
                                                      v
                                                    跳转到仪表盘
```

### 1.2 注册流程

```
用户 -> 访问注册页面
  |
  v
填写注册信息 --> 输入用户名/邮箱/密码 --> 提交
                                        |
                                        v
                                      验证信息
                                        |
                                        v
                                      发送验证邮件
                                        |
                                        v
                                      用户点击验证链接
                                        |
                                        v
                                      账号激活
                                        |
                                        v
                                      跳转到登录页面
```

### 1.3 密码重置流程

```
用户 -> 点击"忘记密码"
  |
  v
输入注册邮箱 --> 提交
                |
                v
              发送重置链接邮件
                |
                v
              用户点击重置链接
                |
                v
              输入新密码
                |
                v
              提交 --> 密码更新
                        |
                        v
                      跳转到登录页面
```

  v

  |                                        |

                  |
                  v

                  v

                  v

## 6. 系统设置流程

### 6.1 用户管理流程

```
管理员 -> 访问用户管理页面
  |
  v
用户操作 --> 创建用户 --> 填写用户信息 --> 提交
  |                                      |
  v                                      v
编辑用户 --> 修改用户信息 --> 提交      创建成功/失败提示
  |                              |
  v                              v
重置密码 --> 确认 --> 发送重置邮件  更新成功/失败提示
  |
  v
启用/禁用用户 --> 确认 --> 更新状态
  |
  v
删除用户 --> 确认 --> 执行删除
```

### 6.2 角色权限管理流程

```
管理员 -> 访问角色权限页面
  |
  v

  |                                                    |
  v                                                    v
编辑角色 --> 修改角色信息 --> 调整权限 --> 提交      创建成功/失败提示
  |                                            |
  v                                            v
删除角色 --> 确认 --> 执行删除                更新成功/失败提示
```

### 6.3 部门管理流程

```
管理员 -> 访问部门管理页面
  |
  v
部门操作 --> 创建部门 --> 填写部门信息 --> 选择上级部门 --> 提交
  |                                                        |
  v                                                        v
编辑部门 --> 修改部门信息 --> 提交                        创建成功/失败提示
  |                              |
  v                              v
移动部门 --> 选择新位置 --> 提交  更新成功/失败提示
  |
  v
删除部门 --> 确认 --> 执行删除
```

### 6.4 系统配置流程

```
管理员 -> 访问系统参数页面
  |
  v

  |                              |
  v                              v

                              v
                            更新成功/失败提示
```
